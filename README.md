# Northstar Studio

A polished, modern one-page studio website built with HTML, CSS, and vanilla JavaScript. The experience includes animated reveals, a scroll progress indicator, responsive layout, and a refined dark/light visual system designed to feel premium and contemporary.

## Highlights
- Modern hero section with layered gradients and glassmorphism styling
- Responsive card-based content sections
- Smooth reveal animations and polished hover states
- Lightweight JavaScript with progressive enhancement

## Run locally
```bash
python3 -m http.server 8000
```
Then open http://localhost:8000 in your browser.
